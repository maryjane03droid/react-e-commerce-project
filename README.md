# React E-Commerce Store

A modern e-commerce web application built with React and Vite that allows users to browse products, view product details, manage a shopping cart, and proceed through checkout in a responsive and intuitive shopping interface.



## Project Information

- **Project Name:** React E-Commerce Store  
- **name** Mary Jane  

## Introduction

This project was developed to simulate a real-world online shopping platform while demonstrating practical frontend development skills using React. It provides users with a smooth shopping experience through responsive design, reusable components, dynamic routing, and centralized state management.


## Project Overview

The application enables users to:

- Browse available products  
- View detailed product information  
- Add/remove items from cart  
- Adjust product quantities  
- Proceed through checkout  
- Access protected routes for restricted pages  

## Target Audience

This application is intended for:

- Online shoppers seeking a digital storefront experience  
- Small businesses needing storefront prototypes  
- Developers and students learning React application architecture  


## User Interface / User Experience

The project emphasizes:

- Clean and modern interface  
- Responsive mobile-friendly design  
- Easy navigation and accessibility  
- Seamless cart and checkout workflow  


## Core Features

### Product Listing
Displays products in a responsive grid layout.

### Product Details
Dedicated page for each product showing:
- Description  
- Price  
- Specifications  
- Reviews  

### Shopping Cart
Users can:
- Add items  
- Remove items  
- Update quantities  
- View cart total  

### Checkout
Protected checkout page for authenticated users.

### Authentication
Basic login functionality with protected routes.
